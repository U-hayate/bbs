# 使い方

- スレッド一覧(index.php)
  - どんなスレッドがあるか見ることができます。
  - スレを立てるボタンを押すとスレッド作成画面へ移動します。
  - 検索を押すと入力した単語がタイトルに含まれるスレッドを表示することができます。
  - タイトルを押すとスレッドを開くことができ、作成者を押すとその人のページへ移動します。
  
- スレッド画面(thread.php)
  - スレッドに書き込まれた投稿を見ることができます。
  - 投稿ボタンを押すと入力した内容が投稿されます。
  - 投稿するにはログインする必要があります。
  - 各投稿のユーザー名、ユーザーIDを押すとその人のページへ移動します。

- スレッド作成画面(thread_create.php)
  - スレッドを作成することができます。
  - 作成するにはログインする必要があります。
  - 確定を押すと入力した内容のタイトルを作成することができます。

- ユーザー情報の閲覧(user.php)
  - ユーザー情報を見ることができます。
  - ログインしている状態かつ自分のページの場合のみ、編集または削除することができます。

- ユーザー情報の編集(usr_edit.php)
  - ユーザー名を編集することができます。

- ユーザー情報の削除(delete.php)
  - アカウント、作成したスレッド、投稿を削除することができます。
  - アカウントを削除した場合、そのアカウントで作成したスレッド及び、投稿がすべて削除されます。
  - 作成したスレッドを削除する場合、そこに書き込まれた投稿がすべて削除されます。

- 会員登録画面(signup.php)
  - ユーザーID、ユーザー名、パスワードを入力して会員登録することができます。
  - 画面下側のルールに従って入力してください。

- ログイン画面(login.php)
  - ユーザーIDとパスワードを入力してログインすることができます。
  - 忘れた場合、ログインすることができなくなるので、必ず覚えておいてください。

- ログアウト画面(logout.php)
  - ログアウトすることができます。
  - ログイン画面から再びログインすることができます。
